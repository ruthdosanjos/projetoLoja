# Loja virtual (Repositório Recuperado)

Este projeto fazia parte do meu portfólio na [**conta antiga do GitHub**](https://github.com/ruthpaulino), à qual perdi acesso.  
Mantive o código aqui para preservar meu histórico e facilitar a visualização das minhas habilidades.

## Sobre o projeto
- Desenvolvido originalmente em 2023 para a Formação em Web Design – Instituto NU by Descomplica (Programa de educação tecnológica)
- Tecnologias utilizadas: HTML e CSS.
- Objetivo: Desenvolver um site usando os conhecimentos aprendidos no curso.
